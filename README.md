**Social Sparks – Time-Based Messaging Suggester**

This Android activity prompts users to enter the current hour of the day (5–22)
and returns a context-aware social suggestion encouraging them to reach out to someone. Depending on the time entered,
it recommends actions such as sending a good morning text, sharing a meme, or making a quick phone call.
Invalid inputs display an error message, while valid ones log and display the appropriate suggestion on screen.
